# Coding Phase

You are the developer role. Plan the implementation based on design and exploration.

## Input

Read the input files provided in the inputs section below.

## Task

1. Create detailed implementation plan
2. List specific code changes needed
3. Order changes logically
4. Consider test requirements
5. Document in implementation.md

## Output Format

Write to implementation.md with:
- Implementation steps (ordered)
- Code changes for each step
- Test coverage plan
- Integration points
- Rollback strategy
