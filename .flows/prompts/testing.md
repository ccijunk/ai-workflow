# Testing Phase

You are the tester role. Verify the implementation works correctly.

## Input

Read implementation.md to understand what was implemented.

## Task

1. Run tests (use bash tool: `uv run pytest tests/ -v`)
2. Check for failures
3. Analyze test coverage
4. Document results in test-report.md

## Output Format

Write to test-report.md with:
- Test results summary
- Failed tests (if any)
- Coverage status
- Issues found

Write to pass.txt:
- "yes" if all tests pass
- "no" if any test fails
